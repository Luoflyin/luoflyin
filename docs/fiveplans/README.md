# Five-year plans White paper

1. 凡是五年计划规划的路线，必须都坚决维护，凡是五年计划规定的指标，必须都坚持始终不渝地执行。
2. 要求在五年计划胜利完成面前要保持清醒头脑，要经受住新一轮五年计划的考验，务必继续地保持谦虚、谨慎、不骄、不躁的作风，务必继续地保持艰苦奋斗的作风。
3. 五年意味着你可以去精熟一门可以傍身的技能，或者在一个学术领域完成系统性的知识储备。
4. 它意味着你可以去思考有没有可能坚持做好一件有价值的事情，甚至把它做到极致。
5. 以五年为期，制定一个长期的目标，做出持久而坚实的努力。
6. 思考一下，如何通过一点一滴的人生增量，完成个人核心竞争力的锻造。
7. 有些人往死里折腾自己，拼死拼活攒下一份家业，那是别人奋斗十代都赶不上的距离，以一世辛劳换数代子孙锦衣玉食，福荫惠及后人，一切都不公平，一切都很公平。
